## Sorting Algorithm
Bubble sort

I like bubble sort because it's one of the simplest algorithms to understand. It's literally built into the name, the bigger number bubbles to the end of the list
### How Bubble Sort Works:
- Compares side by side elements
- Swaps them if they are in the wrong order
- Repeats this process 
- Larger values "bubble" to the end of the list
- Stop when no swaps are needed
---

### Computational Thinking Method 
 - **Pattern Recognition**  
  Noticing that larger values always move toward the end  

- **Abstraction**  
  Focus only on two elements at a time instead of the whole list  

- **Algorithm Design**  
  Using loops to repeatedly process the list until it is sorted

## Design

This app is designed to help users interact with and understand the Bubble Sort algorithm.Users can gernerat a random list use the buton or they could make their own customized list. With whatsoever they choose, they could easily run the algorithm and see the comparisons in real time in the swap logs at the botttom of the app. I also tried to make the logs really comprehensive so the app is doing a bit more education/teaching than just being interactive. The interface is really simple and readable to help aid understanding.
